# Fidakune语言项目 (中文)

Fidakune是一种为全球沟通、信任和协作设计的构造语言，优先考虑认知简单性、发音易用性和文化中立性。

## 核心原则
- **认知简单性**：分析型语法，无例外，主语-动词-宾语 (SVO)。
- **发音易用性**：20个音素，全球通用。
- **表达能力**：通过复合词如`kore-pet`（悲伤）表达细腻情感。
- **文化中立性**：中立手势，如点头表示"是"。
- **协作进化**：语言委员会通过社区输入更新语言。
